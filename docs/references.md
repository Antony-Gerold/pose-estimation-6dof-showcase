# References

Cited in the final report:

1. **P. J. Besl and N. D. McKay** (1992). *A Method for Registration of 3-D Shapes.* IEEE Trans. PAMI, 14(2):239–256. — ICP foundational paper.
2. **N. Correll, K. E. Bekris, D. Berenson, et al.** (2016). *Analysis and Observations from the First Amazon Picking Challenge.* IEEE Trans. Automation Science and Engineering. — APC context and the "robot-grade" framing.
3. **V. Lepetit, F. Moreno-Noguer, P. Fua** (2009). *EPnP: An Accurate O(n) Solution to the PnP Problem.* IJCV, 81(2):155–166. — `solvePnPRansac` underlying method.
4. **C. Rennie, R. Shome, K. E. Bekris, A. F. De Souza** (2016). *A Dataset for Improved RGBD-Based Object Detection and Pose Estimation for Warehouse Pick-and-Place.* IEEE RA-L, 1(2):1179–1185. — The Rutgers APC dataset used throughout.
5. **E. Rublee, V. Rabaud, K. Konolige, G. Bradski** (2011). *ORB: An Efficient Alternative to SIFT or SURF.* IEEE ICCV. — ORB feature detector used for PnP.
6. **G. Bradski** (2000). *The OpenCV Library.* Dr. Dobb's Journal of Software Tools.
7. **M. Dawson-Haggerty et al.** *trimesh: Python library for triangular meshes.* — used for model loading and depth rendering.
8. **F. Pedregosa et al.** (2011). *Scikit-learn: Machine Learning in Python.* JMLR, 12:2825–2830.
